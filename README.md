📌 Sistema de Distribuição de Tarefas
Um sistema simples de gerenciamento e distribuição de tarefas desenvolvido em Python, com foco em lógica de programação, orientação a objetos e simulação de um fluxo real de organização de trabalho em equipe.

🚀 Funcionalidades
👤 Cadastro de usuários
📝 Criação de tarefas
🎯 Definição de prioridade (Baixa, Média, Alta)
✋ Atribuição manual de tarefas
🤖 Distribuição automática (usuário com menos tarefas)
📋 Listagem de usuários e tarefas
💻 Menu interativo no terminal

🧠 Objetivo do Projeto
Este projeto foi desenvolvido com o objetivo de praticar:
Programação Orientada a Objetos (POO)
Estruturas de decisão e repetição
Organização de código
Simulação de sistemas reais de gestão de tarefas
Uso de Git e GitHub para versionamento

🏗️ Tecnologias Utilizadas
Python 3
Enum (para status e prioridade)
Programação Orientada a Objetos

▶️ Como Executar
1. Clonar o repositório
git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
2. Entrar na pasta
cd Distribuidor-de-Tarefas
3. Rodar o sistema
python tarefas.py

💡 Exemplo de uso
===== SISTEMA DE TAREFAS =====
1 - Criar usuário
2 - Listar usuários
3 - Criar e atribuir tarefa manual
4 - Criar e distribuir tarefa automática
5 - Listar tarefas
0 - Sair
